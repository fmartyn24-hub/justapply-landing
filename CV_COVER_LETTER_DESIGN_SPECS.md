# CV & Cover Letter Design Specifications

Based on 2024-2025 professional design best practices research, these specifications guide the creation of Modern, Professional, and ATS template styles for JustApply exports.

## Research Insights

**Key Principles:**
- Recruiters spend ~7 seconds reviewing CVs—design must support quick scanning
- ATS systems are improving but still struggle with complex layouts
- Single-column format is universally safer than multi-column
- White space and hierarchy are critical for readability
- Modern designs work for creative/tech roles; conservative designs for traditional industries

**Sources:** [Resume Trends 2025](https://www.resumeble.com/career-advice/resume-trends), [CV Design Guide](https://www.neuracv.com/resources/blog/the-ultimate-guide-to-elevating-your-cv-design-in-2026), [ATS Friendly Format](https://www.jobscan.co/blog/20-ats-friendly-resume-templates/)

## Template 1: MODERN

**Target Industries:** Tech, creative, marketing, media, startups, forward-thinking companies
**Design Philosophy:** Contemporary, bold, accent-driven, but still ATS-safe

### CV Layout

**Header Section**
- Name: 18-20pt, bold, color = primary accent (medium blue #2563EB)
- Contact info line: 10pt, gray text, left-aligned
  - Format: email | phone | location | LinkedIn | portfolio (as clickable links in DOCX/PDF)
- Optional: Professional photo/avatar (2x2" square, top-right corner)
- Thin accent bar (2pt) in primary color below header
- Generous white space below header (0.3" spacing)

**Section Headers**
- Font size: 12pt, bold
- Color: Primary accent (blue #2563EB)
- Top border: 3pt solid accent color
- Bottom spacing: 0.15"
- All caps optional but recommended for visual impact

**Body Content - Experience Entries**
- Job Title: 11pt, bold, dark gray
- Company: 11pt, regular, primary accent color
- Dates & Location: 10pt, lighter gray
- Bullet points: 10pt, dark gray text
- Line height: 1.3 (slightly open)
- Bullet spacing: 0.05" between bullets
- Entry spacing: 0.2" between entries

**Skills Section**
- Can use subtle background color (#F3F4F6 - light gray) for skills boxes
- Each skill as a tag or inline list
- Font: 10pt

**Education, Certifications, Additional**
- Same hierarchy as experience but slightly compressed
- Font: 10pt for content, 11pt for degree/credential name

**Color Scheme**
- Primary: #2563EB (medium blue) - for headers, accent bar, important info
- Accent: #FF6B35 (orange) - for optional highlights
- Heading text: #1F2937 (dark gray)
- Body text: #374151 (medium gray)
- Background: #FFFFFF (white) - with optional light section backgrounds (#F3F4F6)

**Margins & Spacing**
- Margins: 0.75" all sides
- Section top margin: 0.15"
- Line height: 1.3 for body, 1.2 for headers

### Cover Letter Layout

**Header**
- Date: 10pt
- Recipient address: 10pt
- Line spacing: 1.15

**Opening**
- Greeting: "Dear [Hiring Manager],"
- Opening paragraph: Hook with enthusiasm and specific role interest
- Font: 11pt, 1.15 line spacing

**Body**
- 2-3 paragraphs (never more than 4)
- Each paragraph 3-4 sentences max
- Font: 11pt
- Include: Why you're interested, relevant skills/achievements, cultural fit
- Personalization signals: Mention company details, specific role requirements

**Closing**
- Professional sign-off: "Sincerely,"
- Name
- Optional: Title or tagline

**Color Accent**
- Subtle: Name in header could be in primary color (optional)
- Accent bar after date line (optional, 1pt)

## Template 2: PROFESSIONAL

**Target Industries:** Finance, law, healthcare, government, traditional corporate, academia
**Design Philosophy:** Conservative, minimalist, formal—maximum trustworthiness

### CV Layout

**Header Section**
- Name: 16pt, bold, dark color (navy #1E3A8A or black #1F2937)
- Contact info: 9pt, single line, gray text
  - Format: Email | Phone | Location (no LinkedIn/portfolio unless highly relevant field)
- No photo
- Thin gray line divider (1pt) below header
- Minimal white space (0.2")

**Section Headers**
- Font size: 11pt, bold, all caps
- Color: Dark gray (#1F2937)
- No colored backgrounds or bars
- Underline optional (0.5pt gray line)
- Bottom spacing: 0.1"

**Body Content - Experience Entries**
- Job Title: 10pt, bold, dark gray
- Company & Location: 10pt, regular, dark gray (on same line if possible)
- Dates: 9pt, gray, right-aligned
- Bullet points: 9pt, dark gray, tight spacing
- Bullet spacing: 0.02" between bullets
- Entry spacing: 0.12" between entries (tight)
- Line height: 1.2 (compact)

**Skills Section**
- Comma-separated list format preferred
- Categories in bold, items in regular weight
- Font: 9pt

**Education, Certifications, Additional**
- Compressed, similar to experience
- Font: 9pt

**Color Scheme**
- Primary: #1E3A8A (navy blue) - for name only
- Heading text: #1F2937 (dark gray)
- Body text: #374151 (medium gray)
- Background: #FFFFFF (white, no accents)
- All dividers: Gray (#D1D5DB)

**Margins & Spacing**
- Margins: 0.75" all sides (sometimes 0.6" to fit more content)
- Section top margin: 0.08"
- Line height: 1.2 for body (tighter than Modern)

### Cover Letter Layout

**Header**
- Date: 9pt
- Recipient address: 9pt
- Formal layout with spacing

**Opening**
- Greeting: "Dear Mr./Ms. [Name]," (use name if possible, else "Dear Hiring Manager,")
- Opening paragraph: Professional tone, specific role interest
- Font: 10pt, 1.15 line spacing

**Body**
- 3 paragraphs exactly (professional standard)
- Each paragraph: 3-4 sentences
- Font: 10pt
- Paragraph 1: Why the role interests you
- Paragraph 2: Relevant accomplishments and skills
- Paragraph 3: Cultural fit and enthusiasm
- Minimal personality—focus on qualifications

**Closing**
- Professional sign-off: "Sincerely,"
- Full name only

**Color & Styling**
- No color accents
- Standard fonts only
- Conservative line breaks and spacing

## Template 3: ATS

**Target Industries:** Any (serves as fallback/compatibility format)
**Design Philosophy:** Maximum ATS compatibility—function over form

### CV Layout

**Header Section**
- Name: UPPERCASE, all caps
- Contact information: Separate lines, no special characters
  - Email: [email address]
  - Phone: [phone number]
  - Location: [City, State]
- NO PHOTO
- NO special characters, no dashes or bullets

**Section Headers**
- Font: Plain, regular weight
- Format: ALL CAPS
- Examples: PROFESSIONAL EXPERIENCE, TECHNICAL SKILLS, EDUCATION
- Single blank line above and below each header

**Body Content - Experience Entries**
- Job Title | Company | Location | Dates
- Format: All on one line if possible, separated by |
- Bullet points: Use standard dash (-) only, no special bullets
- Example:
  ```
  - Achieved 40% increase in customer retention through targeted strategies
  - Led team of 5 developers in migrating legacy system to cloud
  ```

**Skills Section**
- Plain format: Category: skill1, skill2, skill3
- Example:
  ```
  Technical Skills: Python, JavaScript, SQL, AWS, Docker
  ```

**Education**
- Bachelor of Science in Computer Science, University Name, 2020
- Or: Degree | School | Graduation Year

**Color Scheme**
- Black and white ONLY
- No colors, no backgrounds, no styling

**Margins & Spacing**
- 1" margins all sides
- Plain text only
- Single line spacing

### Cover Letter Layout

**Header**
- [Date]
- [Blank line]
- [Recipient Name]
- [Company Name]
- [Address]

**Body**
- [Blank line]
- Dear [Name],
- [Blank line]
- Body paragraphs, plain text
- [Blank line]
- Sincerely,
- [Blank line]
- [Your Name]

**Formatting Rules**
- Plain text only
- No special characters
- No bullets (use dashes or numbers)
- Single or 1.15 line spacing
- Standard paragraph breaks

## Implementation Guidelines

### Font Choices
- **Modern:** Calibri or Arial (sans-serif, modern look)
- **Professional:** Georgia or Garamond (serif, traditional) OR Calibri (conservative sans-serif)
- **ATS:** Arial or Courier New (monospace acceptable)

### Font Sizes
- **Modern:**
  - Name: 18-20pt
  - Headers: 12pt
  - Body: 10-11pt
  - Contact info: 10pt

- **Professional:**
  - Name: 16pt
  - Headers: 11pt
  - Body: 9-10pt
  - Contact info: 9pt

- **ATS:**
  - All: 11pt (consistent)

### Page Length
- CV: 1-2 pages (Modern can be longer and more visual; Professional should fit on 1 page when possible)
- Cover Letter: Always 1 page, about 250-400 words

### Structured Data Fields
The export generators should handle these CV sections:
```
Header:
  - Full name
  - Email
  - Phone
  - Location (city, state)
  - Portfolio/website URL
  - LinkedIn URL
  - Photo (Modern only, optional)

Professional Summary: 2-3 sentence overview

Experience:
  - Job title
  - Company
  - Location
  - Dates (start - end)
  - Description
  - Achievements (bullet points)

Skills:
  - Category name
  - Skill items (array)

Education:
  - Degree
  - School/University
  - Graduation date
  - GPA (if strong)

Certifications: Array of certification names

Additional:
  - Awards, publications, languages, etc.
```

### Cover Letter Sections
```
Opening:
  - Engaging hook
  - Role and company name
  - Brief statement of interest

Body Paragraphs (2-3):
  - Relevant experience/achievements
  - Skills that match job
  - Cultural fit, why company appeals to you

Closing:
  - Call to action
  - Professional sign-off
  - Name
```

## ATS Compatibility Checklist

All templates must pass these checks:
- [ ] Single column layout
- [ ] No tables or text boxes
- [ ] Standard section headings
- [ ] Standard fonts (Arial, Calibri, Georgia, Times New Roman)
- [ ] No images (Modern template must handle optional photo gracefully—consider position/format)
- [ ] Clear bullet points (no special characters)
- [ ] Selectable text (not image-based)
- [ ] File format: DOCX or PDF (not Canva or image PDFs)
- [ ] No colored backgrounds (Professional/ATS only)

## Design Evolution Path

1. **Modern Template:** Implement first (most complex, allows for creativity within ATS bounds)
2. **Professional Template:** Second (smaller scope, conservative version of Modern)
3. **ATS Template:** Third (simplest, plain text foundation)

All three should share the same underlying structured data, just rendering it differently based on template rules.
